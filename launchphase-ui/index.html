The Launchphase UI captures a contemporary look and feel that is spacious, bold, and colorful.

Simplified user experience with a unified system that addresses their needs by offering a streamlined and efficent flow.

A design system that humanizes the user's experience with the application and allows for a personable interaction.

What is Launchphase UI?

Launchphase UI is a design system to support the EPC project within Anixter. A contemporary and modern UI that offers a bold look for managing EPC projects.

Why is Launchphase UI Important?

Launchphase UI gives projects a modern look and feel, a delightful user experience, and an easy to use interface for projects. Designed with flexibility in mind, Launchphase UI enables web applications like EPC to be used across a wide spectrum of devices.

Key Features

Bold, consistant and clear design, spacious, accessibility friendly, desktop, tablet, and mobile support, easy to learn and use, cross-browser support, efficient and fast load times.
