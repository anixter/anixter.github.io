Icons

Launchphase UI utilizes the "FontAwesome" library of icons.

  • The library is invoked within the header using: https://use.fontawesome.com/releases/v5.5.0/css/all.css
  • While this library contains hundreds of icons, the icons Launchphase UI uses will be listed below.
